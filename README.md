# Speed typing

## Installation

Using npm (port 5173)

```bash
  git clone https://github.com/NikSerdu/speed-typing-react
  cd speed-typing-react
  npm i
  npm run dev
```

Using pnpm (port 5173)

```bash
  git clone https://github.com/NikSerdu/speed-typing-react
  cd speed-typing-react
  pnpm i
  pnpm run dev
```

Using docker compose (port 80)

```bash
  git clone https://github.com/NikSerdu/speed-typing-react
  cd speed-typing-react
  sudo docker compose up -d
```
